### PCGamingWiki

> Far Cry is a singleplayer and multiplayer first-person adventure, action, FPS and shooter game in the Far Cry series.
> Instructions[6] Go to <path-to-game>\Languages\Movies\[Language installed]\. Replace Crytek.bik, sandbox.bik, and Ubi.bik with empty files. Notes The 64-bit patch will change the file format from .bik to .avi. To skip them, delete or rename the new files with the inclusion of AMD64.
> Fixes AI seeing through tents. Standalone version of GOG fix. (Is not required for the GOG version due to a hotfix added.)
> 32-bit version can be found here. Includes the 64-bit exclusive patches. Adds Quick saving and loading (assigned to the F5 and F9 keys). Includes widescreen FMVs. Large file size due to included videos. FMVs are from the console port, Far Cry Classic, which has altered levels. Cutscenes will have some inconsistencies with the PC version.
> Graphics feature State WSGF Notes Widescreen resolution Vert-. Use Far Cry Widescreen Fix or FarOut Widescreen. Multi-monitor Vert-. Use Far Cry Widescreen Fix or FarOut Widescreen. Ultra-widescreen Vert-. Use Far Cry Widescreen Fix or FarOut Widescreen. 4K Ultra HD Vert-. Use Far Cry Widescreen Fix or FarOut Widescreen. Field of view (FOV) Use FarOut Widescreen or Far Cry Widescreen Fix and set FOVFactor = 1.0 to desired value in FarCry.WidescreenFix.ini. Windowed Borderless fullscreen windowed Use Far Cry Widescreen Fix while in windowed mode. Anisotropic filtering (AF) Up to 8x. Force 16x in the graphic card's control panel.
