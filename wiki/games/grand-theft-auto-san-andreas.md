## Community Notes

### PCGamingWiki

> ### ThirteenAG's Widescreen Fix
> Fixes aspect ratio, HUD and FOV for widescreen and ultrawidescreen monitors. Adds adjustable FOV and HUD/subtitles scaling.
> Installing ThirteenAG's Widescreen Fix[19] Downgrading and ASI Loader required. Download ThirteenAG's Widescreen Fix. Extract the contents of GTASA.WidescreenFix.zip to <path-to-game>. Optionally, a download for high quality textures for HUD and loading screens is available. To install these, download on the same page Widescreen Frontend and copy the models folder from GTASA.WidescreenFrontend.zip to <path-to-game>, replacing the stock files when prompted (or use the modloader)
> Use Improved Fast Loader[20] Place imfast.ini and imfast.asi into <path-to-game>\scripts. Notes ASI Loader required. Can use alongside SilentPatch. Offers the ability to load the last used savegame automatically and disable the loading screen, which can speed up loading times considerably.
> Launch the game with -skip parameter Notes Only works for 1.01 and 2.0 versions.
