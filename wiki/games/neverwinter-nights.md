## Community Notes

### PCGamingWiki

> Edit nwn.ini[3] Open <path-to-game>\nwn.ini. Go to [Display Options]. Add the line Disable Intro Movies=1. Save the changes and close the file.
> Graphics feature State WSGF Notes Widescreen resolution Multi-monitor Ultra-widescreen [5] 4K Ultra HD UI designed for 800x600 and does not scale, rendering game essentially unplayable. UI Scaling for 4k is available in the Enhanced Edition. Field of view (FOV) Windowed See Windowed. Borderless fullscreen windowed AutoHotkey method reported working.[6] Anisotropic filtering (AF) Default amount is 4x, higher levels can be forced through nvidia/amd control panel. Anti-aliasing (AA) 2x, 4x or 8x MSAA. High-fidelity upscaling See the glossary page for potential workarounds. Vertical sync (Vsync) 60 FPS and 120+ FPS No frame rate cap. High dynamic range display (HDR) See the glossary page for potential alternatives. Color blind mode See the glossary page for potential alternatives.
> Keyboard and mouse State Notes Remapping Limited. Cannot rebind hotbar keys for example. Best workaround is autohotkey scripts. Mouse sensitivity Follows windows settings as long as Enable HardwareMouse=1 is set in <path-to-game>\nwn.ini (it is by default). Mouse acceleration Follows Windows settings (called "Enhance pointer precision"). Mouse input in menus Keyboard and mouse prompts Mouse Y-axis inversion Controller Controller support
> Online play requires a CD key; for the GOG.com version, see Multiplayer on the GOG.com version.
