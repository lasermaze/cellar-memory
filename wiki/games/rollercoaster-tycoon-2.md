## Community Notes

### PCGamingWiki

> RollerCoaster Tycoon 2 runs on a slightly improved RollerCoaster Tycoon engine. The core gameplay is identical to the 1999 original - it is an isometric amusement park building game. The game features additions like a scenario editor, new rides and themes, retouched UI, and the ability to build rides and scenery at any elevation, including underground.
> After release, the game received two expansions by Frontier Developments in 2003 - Wacky Worlds and Time Twister. Both added new scenarios, themes and rides which are reskinned variants of existing ones (sometimes with different vehicle stats), but unlike the expansions for the original, they did not change the gameplay in any way.
> While the game has no compatibility issues with modern systems, the community has de facto moved to playing OpenRCT2, an open-source rewrite of this game with many fixes and quality-of-life improvements compared to the original game, which requires its files to run.
> In the Registry Editor, go to HKEY_LOCAL_MACHINE\Software\Wow6432Node\Infogrames\RollerCoaster Tycoon 2 Setup.
> Change the SetupPath string value to point to your game directory (copying the Path value is the easiest way).
